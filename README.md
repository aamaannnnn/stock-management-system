# Stock Management System

A simple and efficient **Stock Management System** developed in **C Programming Language** for small shops and local businesses to manage inventory digitally. The project uses **file handling** to permanently store product details and provides essential inventory operations through a menu-driven interface.

---

## 📌 Project Overview

Manual inventory management is often time-consuming and error-prone for small businesses. This project aims to provide a lightweight and beginner-friendly digital solution that helps shop owners:

* Track stock levels
* Update product details
* Search products instantly
* Detect low stock items
* Store inventory data safely using files

All product data is stored in a text file named:

```text id="kg3gfd"
stock.txt
```

## ⚙️ System Functionalities

Users can enter:

* Product ID
* Product Name
* Quantity
* Price

```text id="1mnctq"

```

---

### 2. Display All Products

Displays all available products in tabular format.

---

### 3. Search Product

Search products instantly using Product ID.

---

### 4. Modify Product

Update:

* Product Name
* Quantity
* Price

---

### 5. Delete Product

Remove products no longer available in stock.

---

### 6. Low Stock Alert

Automatically detects products with quantity less than 5.
